# goinit
Create a Vite-like CLI for Go projects that asks interactive questions and generates a production-ready REST API project.

# Installation

## Using Go

```bash
go install github.com/dev-marees/goinit@latest
```

Make sure your Go bin directory is in your PATH.

```bash
export PATH=$PATH:$(go env GOPATH)/bin
```

Verify the installation:

```bash
goinit --version
```

## Usage

```bash
goinit create
```
